# Towering Above the Rest Archive

To enable this collection, download the full 24-CD archive from this link:
[Download Towering Above the Rest](https://drive.google.com/drive/folders/1BFLIY4va5MpXWtMzc8P-p1ntijUlJcKe?usp=sharing)

Once downloaded, extract your ZIP file directly into this folder. 

Your folder structure should look like this after extraction:
- CD 01 Singles B-Side
- CD 02 Singles B-Side
...

The application will automatically detect the folders and audio files.
